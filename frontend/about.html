Project Overview & Technology Stack

Quantum Screener is a production-grade algorithmic assessment tool designed to mimic the workflow process of institutional investment teams. The backend relies on a highly optimized Unified JSON Prompting Architecture, which allows a single AI inference call to process data through three distinct analytical personas simultaneously, reducing API latency and preventing rate-limit bottlenecks.

Python (FastAPI) Groq API (Llama-3) Yahoo Finance API Chart.js AI EdTech

1. The Bull Persona

Operating within the unified pipeline, this logical segment acts as an optimistic portfolio manager. It consumes raw financial data (Market Cap, Revenue, P/E) in Indian Rupees and is instructed to identify growth potential and argue passionately for the asset's upside.

2. The Bear Persona

Processed in parallel, the Bear Persona plays the role of a risk-averse short seller. It analyzes the exact same dataset but is programmed to strictly identify macroeconomic headwinds, overvaluation indicators, and structural flaws in the company.

3. The Judge Persona

The system's final decision layer. The Judge plays the role of the Chief Investment Officer. It weighs the generated arguments from both the Bull and Bear perspectives against the live market data to output a strict, decisive verdict: BUY, HOLD, or SELL.

Interactive RAG Chat

Once the initial analysis is generated, the platform unlocks a contextual chat interface. Users can pose highly specific questions about the asset's history or market sentiment. The backend maintains the context of the active ticker, offering seamless Retrieval-Augmented Generation functionality.

Quantum Academy (EdTech)

Bridging the gap between complex FinTech and beginner education. This module dynamically generates custom, 3D-animated study flashcards. Users input any financial topic, and the AI agent structures a tailored 20-card deck to teach Indian market fundamentals in real-time.

Dynamic Visualization

The frontend utilizes Chart.js to render the asset's historical price action using smooth, tension-adjusted line charts with dynamic institutional gradients. The AI's final verdict is also mathematically parsed into a responsive Sentiment Doughnut Chart.
